Rendering 3D Objects on the Web

Using Three.JS


With facebooks acquisition of Oculus Rift in mid 2014 and Microsofts recent reveal of it's upcoming HoloLens, there might be more demand for things that a user can actually "feel" and "touch".

With WebGL slowly gaining more robust support in modern web browsers we have the power to render 3D objects using javascript and canvas.

As long as the business need is there and it makes sense financially we should be presenting interesting solutions to our clients that will make them stand out.
